# OTD AI Surfer

Shop: https://OtdAISurfer-gif.github.io/OTDAISURFER-/shop/

This repository contains the OTD AI Surfer project and a simple GitHub Pages shop page at /shop/. If you want to edit the shop page, modify the file shop/index.html.

Getting started

1. Clone the repo:
   git clone https://github.com/OtdAISurfer-gif/OTDAISURFER-.git
2. Create a branch for your work:
   git checkout -b my-feature
3. Make changes, commit, and push:
   git add .
   git commit -m "Your message"
   git push origin my-feature
